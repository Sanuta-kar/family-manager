# Using the In-App Assistant

The Family Mission app includes a chat assistant (powered by OpenClaw). Each person in the family talks to their own assistant personality right inside the app.

> This is **not** the Telegram bot. The Telegram-Codex bridge is a developer tool for editing the codebase and is not part of the family experience. If you are a family member, you only ever use the assistant inside the app.

## What it can do

- Answer questions and chat in a friendly, age-appropriate way.
- Help you set up reminders and missions by drafting them for you.
- Suggest snoozes within the limits a parent has set.

## What it cannot do on its own

The assistant can **propose** changes, but it never changes your schedule by itself. When it drafts something, the app shows a **Confirm card**. Nothing is saved until you tap **Confirm**. If you tap **Reject**, nothing happens.

## Example

You type:

> Remind me to practice piano every day at 18:00

The assistant replies and shows a Confirm card for a daily 18:00 piano reminder. Tap **Confirm** and the reminder is created; it will then alarm each day like any other mission.

## Roles and safety

- **Children** can create only their own reminders. They cannot use the assistant to change or delete missions a parent has protected.
- **Parents** can manage missions for their children and can review chat history for safety. Chat is stored so a parent can see how the assistant is being used.

## If something looks wrong

If the assistant drafts something you did not intend, just tap **Reject** and rephrase. No change is ever applied without your confirmation.
